# 教师质控模块 - 快速上手指南

## 🎯 功能概览

本次为教师用户新增了两个强大的质控分析模块：

### 1️⃣ 质控评价页面
**路径**: `/quality/control-evaluation`

一个可视化的质控数据仪表盘，展示：
- ✅ 完成人数统计（43/43）
- ⏱️ 平均用时（4m21s）
- 📊 平均正确率（91.2%）
- 🤖 AI综合评分（93分）
- 📈 各诊断编码正确率柱状图
- 💡 AI智能修改建议

### 2️⃣ 小组质控页面
**路径**: `/quality/group-control`

六个学习小组的质控对比分析：
- 🎨 精美的小组卡片展示
- 📋 详细的质控报告弹窗
- 🤖 AI生成的改进建议
- 📚 个性化学习资源推荐

## ❓ 常见问题

### Q1: 为什么看不到这两个新页面？
**A**: 请确认您使用的是**教师**或**管理员**账号登录。学生账号无权访问。

### Q2: 图表不显示怎么办？
**A**: 检查是否安装了echarts依赖：
```bash
npm list echarts
```
如果没有安装：
```bash
npm install echarts@5.4.3
```

### Q3: 如何修改模拟数据？
**A**: 直接编辑对应的Vue文件：
- 质控评价: `QualityControlEvaluation.vue` 第120-132行
- 小组质控: `GroupQualityControl.vue` 第180-450行

### Q4: 响应式布局有问题？
**A**: 这两个页面已做了完整的响应式适配，支持从手机到大屏的所有设备。如发现问题，请检查浏览器窗口大小。
