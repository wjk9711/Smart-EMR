### 代码逻辑验证

[userController.ts](file://f:/File/Project/EMR/emr-backend/src/controllers/userController.ts#L241-L275)中的检查逻辑：

```typescript
// 检查每个用户是否有关联数据
for (const user of existingUsers) {
  const assignmentCount = await PatientAssignment.count({ where: { userId: user.id } })
  const patientCount = await InpatientPatient.count({ where: { doctorId: user.id } })
  const recordCount = await InpatientRecord.count({ where: { doctorId: user.id } })
  
  if (assignmentCount > 0 || patientCount > 0 || recordCount > 0) {
    return error(res, 400, `删除失败：用户 "${user.realName || user.username}" 有关联的数据...`)
  }
}
```

这段代码在实际运行中表现良好，成功阻止了有关联数据的用户被删除。
