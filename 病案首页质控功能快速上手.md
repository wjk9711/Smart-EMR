# 病案首页质控功能 - 快速上手

## 🚀 5分钟快速开始

### 第一步：刷新浏览器
```
按 F5 或 Ctrl+R 刷新页面
```

### 第二步：进入病案首页
```
1. 住院管理 → 住院患者
2. 选择一个患者（如：小丽）
3. 点击"病案列表"
4. 点击"新增病案" → "病案首页"
```

### 第三步：添加诊断并质控
```
1. 找到"出院诊断"部分
2. 点击"+ 添加诊断"按钮
3. 填写诊断名称：肺炎
4. 填写疾病编码：J18.9
5. 选择入院病情：1.有
6. 点击该行的"质控"按钮 ⭐
7. 看到提示："编码正确" ✅
```

### 第四步：测试错误编码
```
1. 再添加一个诊断
2. 填写诊断名称：肺炎
3. 填写错误的编码：J20.9
4. 点击"质控"按钮 ⭐
5. 看到警告："编码可能不正确，建议使用: J18.9" ⚠️
6. 点击"应用"自动修正编码
```

### 第五步：手术质控
```
1. 找到"手术及操作"部分
2. 点击"+ 添加手术"按钮
3. 填写手术名称：阑尾切除术
4. 填写手术编码：47.0
5. 点击该行的"质控"按钮 ⭐
6. 看到提示："编码正确" ✅
```

## ❓ 常见问题

### Q1: 点击质控没反应？
**A**: 
1. 检查是否填写了诊断/手术名称
2. 打开浏览器控制台（F12）查看是否有错误
3. 刷新页面重试

### Q2: 建议的编码不对？
**A**: 
1. 当前编码库是示例数据，可能不完整
2. 可以手动输入正确的编码
3. 联系管理员添加缺失的编码

### Q3: 如何批量质控？
**A**: 
1. 当前需要逐个点击质控按钮
2. 未来版本会支持批量质控功能

### Q4: 质控会影响保存吗？
**A**: 
1. 不会！质控只是检查工具
2. 即使质控不通过，也可以保存病案
3. 建议根据质控结果修正后再提交

### Q5: 编码格式是什么？
**A**: 
- **ICD-10**: 1个字母 + 2位数字 + 可选小数点 + 1-2位数字
  - 例如：`J18.9`, `I10`, `E11.9`
- **ICD-9-CM-3**: 2位数字 + 可选小数点 + 1-2位数字
  - 例如：`47.0`, `51.22`, `36.1`

## 🎯 最佳实践

### ✅ 推荐做法
1. **先填名称，后填编码**：让系统智能建议
2. **及时质控**：每填写一个就质控一次
3. **应用建议**：优先使用系统建议的编码
4. **仔细核对**：即使是建议编码也要确认
5. **批量检查**：填写完所有诊断后统一质控一遍

### ❌ 避免做法
1. ~~随意填写编码~~ → 应该使用标准编码
2. ~~忽略质控警告~~ → 应该认真对待每个警告
3. ~~跳过质控步骤~~ → 应该养成质控习惯
4. ~~重复点击质控~~ → 质控一次即可
5. ~~直接复制他人编码~~ → 应该根据实际病情选择

## 🔥 快捷键技巧

虽然当前没有键盘快捷键，但可以使用以下技巧提高效率：

1. **Tab键**：在输入框之间快速切换
2. **Enter键**：确认对话框
3. **Esc键**：取消对话框
4. **鼠标滚轮**：快速滚动表格
