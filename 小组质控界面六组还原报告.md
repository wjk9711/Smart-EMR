## 🔧 技术要点

### 1. Vue响应式数据
```typescript
const currentGroup = ref<any>(null)
```
保存完整的group对象，包含所有字段。

### 2. 条件渲染
```vue
v-if="currentGroup.reportType === 1"
```
根据`reportType`动态显示对应的报告内容。

### 3. 数据驱动
```typescript
{
  reportType: 1,  // 控制显示哪个报告
  reportTitle: '二、病案编码质控结果',  // 显示标题
}
```
通过数据配置控制UI展示，无需硬编码。
