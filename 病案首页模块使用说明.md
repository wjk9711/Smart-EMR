# 病案首页模块使用说明

##  概述

病案首页是住院病案的核心部分，包含患者的基本信息、住院信息、诊断信息、手术信息、费用信息等全面数据。本模块已按照国家标准病案首页格式一比一复刻，支持完整的录入、编辑、预览和打印功能。

| 字段 | 类型 | 说明 |

##  使用流程

### 1. 新增病案首页

**步骤：**
```
住院管理 → 住院患者 → 选择患者 → 病案列表
→ 点击"新增病案"下拉菜单 → 选择"病案首页"
→ 填写病案首页信息 → 保存草稿/提交
```

**操作说明：**
1. 患者基本信息（姓名、性别、年龄、住院号）自动填充且不可修改
2. 病案号自动等于住院号
3. 点击"+ 添加诊断"按钮添加多个出院诊断
4. 点击"+ 添加手术"按钮添加多个手术操作
5. 可以删除不需要的诊断或手术记录
6. 点击"保存草稿"暂存数据（status=draft）
7. 点击"提交"完成病案（status=completed）

### 2. 编辑病案首页

**步骤：**
```
病案列表 → 找到病案类型为"病案首页"的记录
→ 点击"编辑"按钮 → 自动打开病案首页表单
→ 修改信息 → 保存
```

**特点：**
- 系统根据 `record_type='home_page'` 自动识别并打开对应的表单
- 已有数据自动填充到表单中
- 可以添加/删除诊断和手术记录

### 3. 预览病案首页

**步骤：**
```
病案列表 → 找到病案首页记录 → 点击"查看"按钮
→ 弹出预览窗口 → 查看完整信息
```

**功能：**
- 完整展示病案首页所有内容
- 诊断和手术信息以表格形式展示
- 点击"打印"按钮可打印病案首页

### 4. 打印病案首页

**步骤：**
```
预览窗口 → 点击"打印"按钮
→ 浏览器打印对话框 → 选择打印机 → 打印
```

**打印效果：**
- 自动格式化排版
- 表格边框清晰
- 适合A4纸打印

## ⚠️ 注意事项

1. **患者信息保护**
   - 姓名、性别、年龄、住院号等关键字段设为disabled
   - 防止误修改导致数据不一致

2. **病案号规范**
   - 病案号自动等于住院号
   - 保证病案的唯一性和可追溯性

3. **诊断编码**
   - 出院诊断需填写ICD-10编码
   - 建议使用标准编码库

4. **手术记录**
   - 手术级别按照医院规定填写
   - 切口愈合等级格式：切口类别/愈合等级（如：I/甲）

5. **数据完整性**
   - 必填字段会有红色星号标记
   - 提交前会进行表单验证

6. **打印优化**
   - 预览界面提供打印功能
   - 打印样式已优化，适合A4纸
