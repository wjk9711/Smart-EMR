# 质控评价柱状图统一配色方案

## 🎨 配色方案

### 经典蓝绿渐变

**配色代码**：
```typescript
color: new echarts.graphic.LinearGradient(0, 0, 0, 1, [
  { offset: 0, color: '#5470c6' },  // 顶部：深蓝色
  { offset: 1, color: '#91cc75' },  // 底部：浅绿色
])
```

**颜色说明**：
- **顶部 (#5470c6)**：深蓝色，代表专业、稳重
- **底部 (#91cc75)**：浅绿色，代表健康、积极
- **渐变效果**：从上到下的自然过渡，视觉舒适

---

## ✅ 优势

### 1. 统一风格
- 所有12个诊断使用相同的配色
- 避免颜色过多造成视觉混乱
- 符合ECharts默认主题风格

### 2. 专业美观
- 蓝绿渐变是经典的商务图表配色
- 适合医疗质控场景
- 清爽不刺眼

### 3. 易于对比
- 通过柱子高度区分数据差异
- 单胎活产（40.8%）的矮柱子自然突出
- 无需额外颜色标记

### 4. 可访问性
- 色盲友好（蓝绿组合对大多数色盲类型可见）
- 打印友好（灰度转换后仍有对比度）
- 屏幕显示效果好

## 🎯 设计理念

### 为什么选择蓝绿渐变？

1. **ECharts默认主题**
   - `#5470c6` 和 `#91cc75` 是ECharts默认主题色
   - 用户熟悉，认知成本低
   - 与其他ECharts图表风格一致

2. **色彩心理学**
   - 蓝色：专业、信任、稳定
   - 绿色：健康、成长、积极
   - 渐变：动态、流畅、现代

3. **医疗场景适配**
   - 蓝色：医疗行业常用色（医院标识、医护服装）
   - 绿色：生命、健康、康复
   - 符合医疗质控的专业形象

4. **数据可视化最佳实践**
   - 单一色系避免误导
   - 渐变色增加层次感
   - 高度作为主要视觉编码

### 代码片段

```typescript
series: [
  {
    name: '正确率',
    type: 'bar',
    data: diagnosisData.map(d => ({
      value: d.accuracy,
      itemStyle: {
        // ✅ 使用经典蓝色渐变配色，统一风格
        color: new echarts.graphic.LinearGradient(0, 0, 0, 1, [
          { offset: 0, color: '#5470c6' },  // 顶部：深蓝色
          { offset: 1, color: '#91cc75' },  // 底部：浅绿色
        ]),
      },
    })),
    barWidth: '60%',
    label: {
      show: true,
      position: 'top',
      formatter: '{c}%',
      fontSize: 11,
      color: '#333',  // ✅ 标签颜色也统一为深灰色
    },
    // ...
  },
]
```

### 关键参数

| 参数 | 值 | 说明 |
|------|-----|------|
| **渐变方向** | `(0, 0, 0, 1)` | 从上到下垂直渐变 |
| **顶部颜色** | `#5470c6` | 深蓝色 |
| **底部颜色** | `#91cc75` | 浅绿色 |
| **柱子宽度** | `60%` | 适中宽度，不拥挤 |
| **标签颜色** | `#333` | 深灰色，清晰可读 |
